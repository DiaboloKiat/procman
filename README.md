# procman
forked from
https://github.com/RobotLocomotion/libbot/tree/master/bot2-procman

## Build
### Clone lcm
```
$ git clone https://github.com/lcm-proj/lcm
$ cd lcm
$ mkdir build
$ cd build
$ cmake ..
$ make install
```

### Clone procman
```
$ git clone http://github.com/ARG-NCTU/procman
$ cd procman
$ mkdir build
$ cd build
$ cmake ..
$ make install
```
